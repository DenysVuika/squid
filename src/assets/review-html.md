## HTML Code Review Instructions

Analyze the following HTML code and provide constructive feedback focusing on:

**Semantic HTML:**
- Proper use of semantic elements (header, nav, main, article, section, aside, footer)
- Correct heading hierarchy (h1-h6)
- Appropriate use of lists, tables, and forms
- Meaningful element selection over generic divs/spans

**Accessibility:**
- ARIA labels and roles where needed
- Alt text for images
- Proper form labels and input associations
- Keyboard navigation support
- Screen reader compatibility
- Color contrast and text readability

**Best Practices:**
- Valid HTML5 syntax
- Proper nesting and closing of tags
- Consistent indentation and formatting
- Separation of concerns (HTML for structure, CSS for styling, JS for behavior)
- Meta tags for SEO and social sharing

**Performance:**
- Efficient loading strategies (async, defer for scripts)
- Image optimization considerations
- Minimal inline styles and scripts
- Proper resource hints (preload, prefetch)

**Security:**
- Proper escaping of user-generated content
- Safe use of iframes (sandbox attribute)
- Content Security Policy considerations

**Maintainability:**
- Clear class naming conventions
- Logical structure and organization
- Commented sections where helpful
- Reusable components

Provide specific, actionable suggestions for improvement.
